Users

URL Request Method Description
/users GET Gets all users
/users/:id GET Gets user with matching ID
/users/:id DELETE Deletes user with a matching ID
/users/:id PUT Updates user with matching ID
/users/register POST Inserts new user to table and encrypts password
/users/login POST Decrypts password and checks table for matching email and password credentials
